Description

This module contains some useful helper functions that can be used to set various exceptions inside your CORBA methods. It also contains code to map between complex CORBA_Environment exceptions and helpful strings suitable for display to the user.

Typically to test for exceptions the MATECOMPONENT_EX macro is used thus:

matecomponent_exception_get_text ()

char *              matecomponent_exception_get_text           (CORBA_Environment *ev);

Returns a user readable description of the exception. First checks ev against builtin MateComponent exceptions, then falls back to exception names added through matecomponent_exception_add_handler_str or matecomponent_exception_add_handler_fn.
