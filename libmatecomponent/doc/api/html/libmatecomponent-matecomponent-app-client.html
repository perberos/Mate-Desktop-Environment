matecomponent_app_client_new ()

MateComponentAppClient *   matecomponent_app_client_new               (MateComponent_Application app_server);

app_server: object reference to a MateComponent::Application; this function takes ownership of this reference (use matecomponent_object_dup_ref() if you want to keep your own reference.)

Create an application client object connected to the remote (or local) MateComponent::Application object.

Warning

Applications should not use this function. See matecomponent_application_register_unique().

matecomponent_app_client_msg_send ()

GValue *            matecomponent_app_client_msg_send          (MateComponentAppClient *app_client,
                                                         const char *message,
                                                         CORBA_Environment *opt_env,

Send a message to the application server. Takes a variable length argument list of GType, value pairs, terminated with G_TYPE_NONE. Values are direct C values, not GValues! Example:

                                     G_TYPE_BOOLEAN, TRUE,
                                     G_TYPE_NONE);
